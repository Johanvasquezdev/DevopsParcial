GitHub Actions

Workflow automatico que se activa en cada push. Instala dependencias y despliega sin intervension manual.

🚀

Surge.sh

Hosting estatico gratuito y ultra-rapido. El sitio queda en linea en segundos con un dominio personalizado.

🔒

Secrets Seguros

El token de Surge se guarda como secreto cifrado en GitHub. Nunca queda expuesto en el repositorio.

# .github/workflows/main.yml
name: Deploy to Surge.sh

on:
  push:
    branches: [main]

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - run: npm install -g surge
      - run: surge ./ johan-cicd.surge.sh
        env:
          SURGE_TOKEN: ${{ secrets.SURGE_TOKEN }}
